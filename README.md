# @danielbiegler/bunny-file-storage

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.6. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.

## Tests

```bash
bun run test
```

For integration tests valid credentials are required, see `.env.example` copy them over to `.env` after you get them.
